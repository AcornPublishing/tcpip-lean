This is the ChipWeb™ server, running on a Microchip PIC16F877 microcontroller. It includes
  • Miniature Ethernet Web server
  • Stateless TCP/IP stack
  • Pages stored in 32 Kbyte ROM, programmed via serial link
  • Dynamic Web page graphics for analogue and digital I/O
The status bar on the right is updated every few seconds. For more information, click the links on the left.
